# 🍎 Guess the Fruit Name

A fun, classic word-guessing game (Hangman-style) written in Python! Test your luck and deduction skills by guessing the hidden fruit name letter by letter before you run out of lives.

## 📖 Description

This is a lightweight, command-line game where the computer picks a random fruit from a predefined list, and you have to guess what it is. You are given a set number of lives, and for every incorrect letter you guess, you lose one. Can you uncover the fruit before your lives run out?

## ✨ Features

- 🍉 **Diverse Fruit List**: The game randomly selects from a wide variety of fruits (apple, banana, strawberry, watermelon, kiwi, and more!).
- ❤️ **Lives System**: You start each round with **6 lives**. 
- ⚠️ **Penalty for Repeats**: If you guess a letter you've already tried, you will lose a life!
- 💡 **Length Hint**: The game tells you exactly how many letters are in the secret fruit's name to help you get started.
- 🔁 **Endless Replayability**: Once a round ends (win or lose), you can instantly start a new one.
- 🐍 **Zero Dependencies**: Built entirely with Python's standard library (`random`). No `pip install` required!

## 🛠️ Prerequisites

- Python 3.x installed on your system.
